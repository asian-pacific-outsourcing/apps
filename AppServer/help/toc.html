Application Server Help

Application Server is a secure server configurable to run in various blocking or non-blocking modes to service clients over the Internet or local network.

Start Up

Server is configured at start up to run in one of the following modes:

  • B1 Blocking/Single-threaded Server
  • BN Blocking/Multi-threaded Server
  • BP Blocking/Pooled-Thread Server
  • N1 Nonblocking/Single-threaded Server
  • N2 Nonblocking/Dual-threaded Server
In addition to the mode, the following settings are specified once at start up:
  • Port The physical port number upon which this servers receives request
  • x509 Whether the server should run in secure mode (requires an X509 Certificate)
